# Card Component

## Props
- `padded` — default true, adds `--space-6` (24px) internal padding
- `elevated` — default true, applies brand soft blue-tinted shadow (`--shadow-md`)
- `style` — override for custom contexts

## Conventions
- Background: `--color-bg-surface` (white) on linen page background
- Border: `--border-thin` when not elevated
- Radius: `--radius-lg` (14px)
- No neumorphism — single soft outer shadow only
- Shadow tint always uses brand blue base, never neutral gray
